## О проекте
Toolify - это интернет-магазин хозяйственных товаров. 
Веб-сайт позволяет пользователям выбирать товары, оформлять заказы, а администраторам - управлять пользотвателями, ассортиментом и заказами.
## Основной функционал
**Для Пользователей:**
- регистрация и авторизация;
- личный кабинет редактирование данных, история покупок, отслеживание состояний заказов;
- Каталог товаров с фильтрацие по категорям, поиском, просмотром карточек товара;
- Корзина добавление, изменение количества, удаление товаров;
- Оформление заказа: указание контактных данных, адреса доставки, способа оплаты и доставки.
- История заказов и отслеживание статусов.

**Для администратора:**
- Добавление/удаление и редактирование товаров;
- Управление категориями товаров;
- Удаление пользователей;
- Просмотр всех заказов пользователей;
- Изменение статусов заказов;
- Изменение количества товаров.
